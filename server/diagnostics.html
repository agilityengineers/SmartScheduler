Common Issues & Solutions

EENVELOPE Error

The recipient address is being rejected by the mail server.

Solutions:
  • Use a valid email address for testing
  • Check for proper DNS entries for the domain
  • Avoid using temporary or disposable email addresses
  • Verify SPF, DKIM, and DMARC records

Authentication Failures

SMTP server rejects the login credentials.

Solutions:
  • Verify SMTP_USER and SMTP_PASS are correct
  • Check if the email provider requires app passwords
  • Ensure the account has sending permissions
  • Check for rate limits or sending restrictions

Connection Issues

Cannot establish connection to the SMTP server.

Solutions:
  • Verify SMTP_HOST is correct
  • Ensure SMTP_PORT and SMTP_SECURE match (465=secure, 587=TLS)
  • Check for firewall restrictions
  • Verify network connectivity to the mail server

Email Not Delivered

Email appears to send successfully but is not received.

Solutions:
  • Check spam/junk folders
  • Verify sender domain has proper DNS records
  • Use a FROM_EMAIL that matches the SMTP server domain
  • Try with different email providers (Gmail, Outlook, etc.)
